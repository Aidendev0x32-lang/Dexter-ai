# `dexter tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
dexter tui
dexter tui --url ws://127.0.0.1:18789 --token <token>
dexter tui --session main --deliver
```
